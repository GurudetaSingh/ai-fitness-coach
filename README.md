# 🏋️ IronMind — AI Fitness Tracker & Coach

IronMind is a modern fitness-tracking web app that helps users log workouts, visualize their progress, and receive AI-driven coaching insights.

---

## 🚀 Features

* Log workouts (exercise, weight, reps, sets, date)
* Track progress with charts (weight + volume)
* View stats (total volume, max weight, active days)
* AI insights:

  * Detect plateaus
  * Identify fatigue trends
  * Suggest improvements (deloads, variety, recovery)

---

## 🛠️ Tech Stack

* React 18 + TypeScript
* Vite
* Tailwind CSS
* Recharts
* Radix UI
* localStorage

---

## ⚙️ How It Works

* Workout data is stored in localStorage and managed with React state
* Charts visualize progress by exercise over time
* A rule-based AI engine analyzes trends (plateaus, volume changes, frequency) and generates coaching insights
